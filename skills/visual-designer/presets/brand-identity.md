# Brand Identity Preset

Lightweight brand guidelines for presentations, documents, marketing materials, and communications. No UI components — focused on visual language and tone.

## 1. Color Palette

### Primary Palette (6 colors)

| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| Primary | #___ | rgb(_,_,_) | Brand identity, headlines, CTAs |
| Primary Light | #___ | rgb(_,_,_) | Backgrounds, hover states |
| Primary Dark | #___ | rgb(_,_,_) | Text on light backgrounds |
| Secondary | #___ | rgb(_,_,_) | Accents, supporting elements |
| Neutral Dark | #___ | rgb(_,_,_) | Body text, headings |
| Neutral Light | #___ | rgb(_,_,_) | Backgrounds, borders |

### Usage Rules
- Primary color: maximum 20% of any layout
- Always pair primary with sufficient contrast (4.5:1 for text)
- Use neutral dark for body text, never pure black (#000)
- Secondary color for charts, diagrams, highlights — not for text
- On dark backgrounds, use Primary Light, not Primary

## 2. Typography

### Font Selection

| Role | Font | Fallback | Usage |
|------|------|----------|-------|
| Display | [Selected] | system sans-serif | Titles, hero text |
| Body | [Selected] | system sans-serif | Paragraphs, notes |
| Code | [Selected mono] | monospace | Code snippets |

### Type Scale

| Level | Size | Weight | Usage |
|-------|------|--------|-------|
| Display | 48-64pt | Bold | Presentation titles, hero slides |
| Headline | 32-40pt | Bold | Section headers |
| Title | 24-28pt | Semibold | Slide titles, document headers |
| Subtitle | 18-22pt | Medium | Subtitles, callouts |
| Body | 16-18pt | Regular | Main content |
| Caption | 12-14pt | Regular | Labels, footnotes, sources |

### Typography Rules
- Maximum 2 font families
- Headlines: Bold or Semibold only
- Body: Regular weight, never Light for small sizes
- Minimum size for readability: 12pt print, 16px screen, 18pt projection
- Line height: 1.4-1.6 for body text

## 3. Imagery & Photography

### Style Direction
Based on personality, define:
- **Mood:** [light & airy / dark & moody / vibrant / muted / editorial]
- **Subject matter:** [people / abstract / product / landscape / conceptual]
- **Composition:** [centered / rule of thirds / asymmetric / minimal negative space]
- **Post-processing:** [natural / high contrast / desaturated / warm tint / cool tint]

### Image Rules
- All images must have sufficient resolution (min 150 DPI for print, 72 DPI for screen)
- Maintain consistent color temperature across all imagery
- People photography: diverse, authentic, no obvious stock poses
- Abstract/pattern usage: tie to brand color palette

### Iconography
- **Style:** line / filled / duotone
- **Weight:** consistent stroke width (1.5-2px recommended)
- **Grid:** design on 24x24 base grid
- **Color:** single color from brand palette, or neutral

---

## 4. Logo Usage

### Clear Space
- Minimum clear space: height of the logo mark on all sides
- Never place logo on busy backgrounds without a container

### Sizing
- Minimum size: define for print and screen
- Preferred placement: top-left for documents, centered for presentations

### Don'ts
- Don't stretch, rotate, or skew
- Don't change colors outside approved palette
- Don't add effects (shadows, outlines, gradients)
- Don't place on low-contrast backgrounds
